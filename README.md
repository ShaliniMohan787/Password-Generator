# Password-Generator
Secure Password Generator is a modern web application that creates strong, customizable passwords using cryptographically secure randomness. It includes a strength meter, entropy calculation, crack-time estimation, password history, dark mode, and a responsive cybersecurity-style interface.
#  Secure Password Generator

A modern cybersecurity-focused password generator built using HTML, CSS, and JavaScript.

## Features

- Cryptographically secure password generation
- Adjustable password length
- Uppercase, lowercase, numbers, and symbols options
- Password strength meter
- Entropy calculator
- Crack time estimation
- Copy to clipboard
- Password history
- Download password as TXT
- Dark mode
- Responsive modern UI

## Technologies

HTML  
CSS  
JavaScript

## Security

Passwords are generated locally using the browser's cryptographic random generator (`crypto.getRandomValues`) and are never stored permanently.
